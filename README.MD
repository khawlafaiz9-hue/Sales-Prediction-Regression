# 📊 Sales Prediction & Regression Models

مشروع يهدف إلى التنبؤ بنسب التحويل (Conversion Rate) وتحليل العلاقة بين الإنفاق التسويقي وأداء المبيعات باستخدام نماذج الانحدار.

## 🛠️ التقنيات المستخدمة
* **Python**
* **Scikit-Learn** (بناء نماذج Linear Regression و Decision Tree Regressor)
* **Matplotlib & Pandas** (تحليل البيانات والرسوم البيانية)
* **Performance Metrics** (حساب R2 Score و MSE)

## 📌 أهداف المشروع
* معالجة وتحليل بيانات التسويق الإلكتروني.
* مقارنة دقة نماذج الانحدار المختلفة في التنبؤ بالمبيعات.
* تحديد العوامل الأكثر تأثيراً في زيادة رضا العملاء ونسب التحويل.
